Suspend on room setpoint

enable
When enabled, the heating circuit will be suspended (turned off) when the measured room temperature reaches or exceeds the setpoint + offset.
  • Prevents unnecessary heating when the room is already warm enough.
  • Uses a hysteresis and offset to avoid rapid toggling.
  • Not recommended for condensing boilers or heating pumps.
Tip: Adjust the hysteresis and suspend offset below to fine-tune when heating resumes after suspension.

Heating curve

num. of points
Gradient
Defines how strongly the target flow temperature rises as outside temperature falls.
  • Higher gradient: steeper heating curve and higher flow temperatures in cold weather
  • Lower gradient: flatter heating curve and lower flow temperatures
  • Increase it if rooms stay too cold in winter
  • Decrease it if the system overheats when outside temperatures drop

Source: www.bosy-online.de

Offset
Shifts the entire heating curve up or down by a fixed amount.
  • Positive offset: increases target flow temperature across the full curve
  • Negative offset: decreases target flow temperature across the full curve
  • Use this for fine adjustment when the overall heating level is slightly too warm or too cool
Unlike Gradient, Offset changes the curve evenly instead of making it steeper or flatter.
K
Returnlimit ΔT
If the actual system temperature is above the return heatingcurve, a PI controller throttles the flow temperature.
  • 0 K: disabled
  • Higher value: stronger return-temperature limiting
  • Typical starting point: 3 to 5 K
Use the smallest value that still improves condensing operation and system stability.
